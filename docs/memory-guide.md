## 功能概述

MSA 记忆系统是一个本地持久化的记忆管理功能，它可以：

- **自动记录聊天会话** - 每次对话都会被保存
- **AI 知识提取** - 从对话中自动提取用户画像、概念、策略等知识
- **智能记忆注入** - AI 会"记住"你的偏好和历史，提供个性化回复
- **强大的搜索能力** - 快速搜索历史会话和知识库
- **安全的隐私保护** - 自动过滤敏感信息（API Key、密码等）

## 快速开始

### 第一次使用

1. 启动 MSA：
   ```bash
   ./msa
   ```

2. 记忆系统会自动初始化，无需额外配置

3. 开始正常聊天，系统会在后台自动记录和提取知识

### 打开记忆浏览器

在聊天界面输入命令：
```
/remember
```

你将看到记忆浏览器主页：
```
📚 记忆浏览器

 [1] 📝 历史会话
     查看和管理历史聊天记录

 [2] 🧠 知识库
     查看从对话中提取的知识

 [3] 🔍 搜索记忆
     在会话和知识中搜索

 [4] 📊 统计信息
     查看记忆系统的使用统计

ESC/q: 返回聊天 | ↑/↓: 导航 | Enter: 选择
```

## 查看历史会话

### 进入会话列表

1. 在记忆主页选择 `1` 或 `历史会话`
2. 按数字键 `1` 或用 `↑/↓` 选择后按 `Enter`

### 会话列表功能

会话列表显示所有历史会话，按时间倒序排列：

```
📝 历史会话

/search...                           (按 / 进入搜索模式)

```

### 快捷键

| 按键 | 功能 |
|------|------|
| `↑` / `k` | 上移 |
| `↓` / `j` | 下移 |
| `PgUp` | 向上翻页 |
| `PgDn` | 向下翻页 |
| `/` | 进入搜索模式 |
| `d` | 删除选中的会话 |
| `Enter` | 查看会话详情 |
| `ESC` | 返回主页 |

1. 在会话列表中选中一个会话

### 会话详情快捷键

| 按键 | 功能 |
|------|------|
| `↑` / `k` | 向上滚动一行 |

1. 在会话列表中选中要删除的会话
2. 按 `d` 键删除
3. 会话将立即从列表中移除，文件被删除

## 使用知识库

### 进入知识库

1. 在记忆主页选择 `2` 或 `知识库`
2. 按数字键 `2` 或用 `↑/↓` 选择后按 `Enter`

### 知识类型

知识库包含 6 种类型的知识：

```
🧠 知识库

 [1] 👤 用户画像         (2 条)
 [2] 📊 关注列表         (5 条)
 [3] 📚 学过的概念       (12 条)
 [4] 📋 交易策略         (3 条)
 [5] ❓ 常见问答         (8 条)
 [6] 🔍 市场洞察         (1 条)

ESC/q: 返回聊天 | ↑/↓: 导航 | Enter: 选择
```

### 知识类型说明

| 类型 | 说明 | 示例 |
|------|------|------|
| 👤 用户画像 | AI 学习到的你的投资风格、风险偏好等 | 风险偏好: 稳健、投资风格: 价值投资 |
| 📊 关注列表 | 你关注的股票和板块 | 腾讯控股、阿里巴巴、新能源板块 |
| 📚 学过的概念 | 你询问过的技术指标、术语等 | RSI、MACD、布林带 |
| 📋 交易策略 | 讨论过的交易策略和技巧 | 金叉策略、突破买入法 |
| ❓ 常见问答 | 常见问题的回答 | 如何查询股价、如何设置止损 |
| 🔍 市场洞察 | 从对话中提取的市场见解 | 新能源板块趋势分析 |

### 查看知识

1. 选择知识类型（按 1-6 或用方向键选择）
2. 按 `Enter` 查看该类型的所有知识

示例：查看"学过的概念"

## 搜索记忆

### 进入搜索

1. 在记忆主页选择 `3` 或 `搜索记忆`
2. 或直接按数字键 `3`

### 搜索界面

```
🔍 搜索记忆

/腾讯_                                 (过滤: 全部 Tab 切换)

输入搜索词并按 Enter 搜索
↑/↓: 浏览历史

ESC/q: 退出 | Tab: 切换过滤 | Enter: 搜索
```

### 搜索功能

- **输入搜索词**：直接输入关键词
- **浏览历史**：按 `↑/↓` 浏览之前的搜索
- **切换过滤**：按 `Tab` 在以下过滤类型间切换：
  - `全部` - 搜索会话和知识
  - `会话` - 只搜索会话
  - `知识` - 只搜索知识

### 搜索结果

```
🔍 搜索结果

/腾讯                                    过滤: 全部 [1/3]

▶ [2024-03-13 14:30] 💬 如何查询腾讯股价
   腾讯控股(00700.HK)的实时股价可以通过...

▶ [2024-03-12 09:15] 🧠 关注列表
   - 腾讯控股 (长期关注)
   - ...

 1/3                                       ↑/↓: 导航 | Enter: 打开 | ESC: 返回搜索
```

### 搜索结果操作

- **↑/↓** - 选择结果
- **Enter** - 打开选中项（会话或知识）
- **ESC** - 返回搜索输入

## 隐私与安全

### 敏感信息保护

记忆系统会自动过滤和脱敏以下敏感信息：

- **API Key** - `sk-abc123...` → `***REDACTED***`
- **Bearer Token** - `Bearer xxx...` → `***REDACTED***`
- **密码** - `password:123456` → `***REDACTED***`

### 数据存储位置

所有数据存储在本地：
```
~/.msa/remember/
```

- **不会**上传到云端
- **不会**与第三方共享
- **完全由你控制**

### 清空记忆

如果需要清空所有记忆：

```bash
# 方法 1: 删除数据目录
rm -rf ~/.msa/remember/

# 方法 2: 设置环境变量禁用后删除
MSA_MEMORY_ENABLED=false ./msa
rm -rf ~/.msa/remember/
```

## 禁用记忆系统

### 方法 1: 环境变量（推荐）

设置环境变量：
```bash
export MSA_MEMORY_ENABLED=false
./msa
```

### 方法 2: 临时禁用

```bash
MSA_MEMORY_ENABLED=false ./msa
```

### 禁用后的行为

- 不记录聊天会话
- 不提取知识
- 不注入历史知识到 AI
- `/remember` 命令仍可用但无数据

## 常见问题

### Q: 记忆系统会影响性能吗？

A: 影响很小：
- 启动时间增加 < 200ms
- 内存占用增加 < 20MB
- 知识提取在后台异步执行，不阻塞退出

### Q: AI 如何使用我的记忆？

A: AI 会在回复时自动获取以下知识：
- 你的投资风格和风险偏好
- 你关注的股票
- 你学过的概念和术语
- 常见问题的答案

### Q: 可以编辑或删除记忆吗？

A:
- **删除会话**：在会话列表中按 `d` 键
- **清空所有记忆**：删除 `~/.msa/remember/` 目录
- **编辑功能**：正在开发中

### Q: 记忆会占用多少磁盘空间？

A: 估算：
- 每个会话约 10-50KB
- 100 个会话约 1-5MB
- 1000 个会话约 10-50MB

### Q: 可以导出记忆数据吗？

A: 导出功能正在开发中，届时支持导出为：
- JSON 格式（机器可读）
- 文本格式（人类可读）

### Q: AI 提取的知识准确吗？

A:
- 知识提取使用 AI LLM，准确率较高
- 系统使用降级策略，AI 失败时使用规则提取
- 你可以在知识库中查看提取的结果
- 如果有错误的知识，可以通过删除会话来移除

### Q: 如何查看 AI"记住"了什么？

A:
1. 输入 `/remember` 进入记忆浏览器
2. 选择 `2` 查看知识库
3. 浏览各类知识，AI 就是根据这些知识来个性化回复的

## 技巧与窍门

### 快速搜索

- 在会话列表中按 `/` 进入搜索模式
- 可以搜索股票代码、日期、关键词

### 键盘导航

- 使用数字键 `1-4` 快速访问主菜单
- 使用 `j/k` 代替 `↑/↓`（vim 风格）
- 使用 `G` 快速跳到最后

### 搜索历史

- 在搜索界面按 `↑/↓` 浏览之前的搜索
- 搜索历史会保存在当前会话中

### 退出时保存

- 按 `Ctrl+C` 或 `ESC` 正常退出会自动保存
- 如果强制终止（如 `kill -9`），会尽力保存已记录的消息
