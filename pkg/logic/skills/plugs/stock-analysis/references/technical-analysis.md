# 技术面分析框架

## 必须覆盖的内容

### 1. 趋势分析

- 日线级别处于上升趋势/下降趋势/横盘震荡
- 均线系统状态（5日、10日、20日、60日均线）
- 趋势线支撑/阻力

### 2. 支撑阻力位

- 关键支撑位：具体价格
- 关键阻力位：具体价格
- 支撑/阻力的来源（前高前低、均线、整数关口）

### 3. 成交量分析

- 近 3 日成交量较上周的变化
- 量价配合情况
- 是否出现量价背离

### 4. 技术指标（至少分析 2 个）

#### MACD

```
- DIF 和 DEA 的位置关系
- 金叉/死叉信号
- 柱状图变化趋势
- 是否出现背离
```

#### RSI

```
- 当前 RSI 数值
- 是否进入超买(>70)或超卖(<30)区域
- 是否出现背离
```

#### KDJ

```
- K、D、J 三线位置
- 金叉/死叉信号
- 是否进入超买/超卖区域
```

#### 布林带

```
- 股价在布林带中的位置
- 布林带收窄/开口状态
- 是否触及上下轨
```

---

## 技术面信号判断

### 看涨信号

| 信号 | 权重 |
|------|------|
| 股价站上所有均线 | 高 |
| MACD 金叉 | 中 |
| RSI 从超卖区回升 | 中 |
| 放量突破阻力位 | 高 |
| 布林带下轨反弹 | 低 |

### 看跌信号

| 信号 | 权重 |
|------|------|
| 股价跌破关键均线 | 高 |
| MACD 死叉 | 中 |
| RSI 进入超买区后回落 | 中 |
| 放量跌破支撑位 | 高 |
| 布林带上轨回落 | 低 |

```markdown
#### 趋势判断
[上升趋势/下降趋势/横盘震荡]

#### 关键价位
- 支撑位：[价格] ([来源])
- 阻力位：[价格] ([来源])

#### 成交量
[成交量分析结论]

#### 技术指标
- MACD：[分析结论]
- RSI：[分析结论]

#### 技术面结论
[综合判断，看涨/看跌/中性]
```
