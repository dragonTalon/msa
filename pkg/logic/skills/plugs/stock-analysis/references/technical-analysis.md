# 技术面分析框架

## 必须覆盖的内容

### 1. 趋势分析

- 日线级别处于上升趋势/下降趋势/横盘震荡
- 均线系统状态（5日、10日、20日、60日均线）
- 趋势线支撑/阻力

### 2. 支撑阻力位

- 关键支撑位：具体价格
- 关键阻力位：具体价格
- 支撑/阻力的来源（前高前低、均线、整数关口）

### 3. 成交量分析

- 近 3 日成交量较上周的变化
- 量价配合情况
- 是否出现量价背离

### 4. 技术指标（至少分析 3 个）

#### MACD

```
- DIF 和 DEA 的位置关系
- 金叉/死叉信号
- 柱状图变化趋势
- 是否出现背离
```

#### RSI

```
- 当前 RSI 数值
- 是否进入超买(>70)或超卖(<30)区域
- 是否出现背离
```

#### KDJ

```
- K、D、J 三线位置
- 金叉/死叉信号
- 是否进入超买/超卖区域
```

#### 布林带

```
- 股价在布林带中的位置
- 布林带收窄/开口状态
- 是否触及上下轨
```

#### VWAP (Phase 3 新增)

```
- VWAP = Σ(Price_i × Volume_i) / Σ(Volume_i)
- 日内精确 VWAP: 基于 get_stock_minute_k 分钟数据
- 日线近似 VWAP: 基于日线 (H+L+C)/3 替代精确价格
- 当前价 > VWAP: 多头占优，机构成本之上
- 当前价 < VWAP: 空头占优，机构成本之下
- ⚠️ 日线近似值精度有限，标注 "VWAP 近似值"（如使用日线）
```

#### ATR(14) (Phase 3 新增)

```
- 引用 trading-common/references/risk-management.md 的完整计算方法
- ATR/价格: >3% 高波动, 1-3% 正常, <1% 低波动
- 高波动 → 宽止损、小仓位
- 低波动 → 紧止损、可伺机突破
```

#### OBV (Phase 3 新增)

```
- 若收盘 > 昨收: OBV = 前日OBV + 当日成交量
- 若收盘 < 昨收: OBV = 前日OBV - 当日成交量
- 数据来源: get_stock_history_k (日线 close + volume)
- OBV 与价格同向: 趋势确认
- OBV 与价格背离: 趋势反转预警
```

#### 多时间周期框架 (Phase 3 新增)

```
三层周期:
- 周线: 长期趋势方向 (使用 get_stock_history_k period="week")
  - 均线多头 → 倾向做多; 均线空头 → 倾向做空/观望
- 日线: 中期入场信号 (使用 get_stock_history_k period="day")
  - MACD/RSI/支撑阻力 在周线方向上寻找入场点
- 60分钟: 短期精确入场 (使用 get_stock_minute_k 聚合)
  - 将分钟数据按 60 分钟分组: Open=首价, Close=末价, High=最高, Low=最低

多周期共振判断:
| 周线 | 日线 | 60分钟 | 结论 |
|------|------|--------|------|
| 多头 | 多头 | 多头 | 强买入 |
| 多头 | 多头 | 空头 | 等60分钟转多 |
| 多头 | 空头 | - | 不买入 |
| 空头 | - | - | 不买入 |
```

## 技术面信号判断

### 看涨信号

| 信号 | 权重 |
|------|------|
| 股价站上所有均线 | 高 |
| MACD 金叉 | 中 |
| RSI 从超卖区回升 | 中 |
| 放量突破阻力位 | 高 |
| 布林带下轨反弹 | 低 |

### 看跌信号

| 信号 | 权重 |
|------|------|
| 股价跌破关键均线 | 高 |
| MACD 死叉 | 中 |
| RSI 进入超买区后回落 | 中 |
| 放量跌破支撑位 | 高 |
| 布林带上轨回落 | 低 |

```markdown
#### 关键价位
- 支撑位：[价格] ([来源])
- 阻力位：[价格] ([来源])

#### 成交量
[成交量分析结论]

#### 技术指标
- MACD：[分析结论]
- RSI：[分析结论]

#### 技术面结论
[综合判断，看涨/看跌/中性]
```
