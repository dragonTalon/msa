# 市场面分析框架

## 必须覆盖的内容

### 1. 资金流向

#### 北向资金

- 近 5 日净买入金额
- 当日净买入/净卖出
- 北向资金持股变化趋势

#### 主力资金

- 主力净流入/流出金额
- 大单、中单、小单分布
- 连续流入/流出天数

#### 融资融券

- 融资余额变化
- 融券余额变化
- 融资买入额占比

### 2. 市场情绪

#### 板块表现

- 所属行业板块涨跌幅
- 板块内个股表现
- 板块资金流向

#### 市场热度

- 换手率
- 量比
- 涨停/跌停家数

### 3. 政策影响

- 相关政策动态
- 政策对行业的影响
- 未来政策预期

### 4. 市场事件

- 近期重大公告
- 行业事件
- 宏观经济数据

---

## 资金流向判断

### 主力行为分析

| 现象 | 解读 |
|------|------|
| 主力连续大幅流入 | 主力吸筹，看涨 |
| 主力连续大幅流出 | 主力出货，看跌 |
| 主力小幅流入，股价下跌 | 主力洗盘，可能见底 |
| 主力小幅流出，股价上涨 | 散户追涨，风险加大 |

### 北向资金分析

| 现象 | 解读 |
|------|------|
| 北向资金持续流入 | 外资看好，中长期利好 |
| 北向资金持续流出 | 外资撤离，中长期利空 |
| 北向资金逆势买入 | 价值投资机会 |

## 分析输出格式

```markdown
### 市场面分析

#### 资金流向
- 北向资金：近5日净买入 xx 亿元
- 主力资金：今日净流入/流出 xx 亿元
- 融资余额：xx 亿元，较昨日 +/-

#### 市场情绪
- 板块表现：[行业板块] 涨跌幅 xx%
- 换手率：xx%
- 市场热度：[高/中/低]

#### 政策与事件
- [相关政策或事件]
- [对股价的影响分析]

#### 市场面结论
[综合判断，积极/中性/消极]
```
