# 基本面分析框架

## 必须覆盖的内容

### 1. 财务数据

#### 盈利能力

| 指标 | 说明 | 判断标准 |
|------|------|----------|
| 净利润同比增长 | 反映盈利增长 | >20% 为优秀 |
| 毛利率 | 反映产品竞争力 | 行业对比 |
| 净利率 | 反映经营效率 | 行业对比 |
| ROE | 净资产收益率 | >15% 为优秀 |

#### 偿债能力

| 指标 | 说明 | 判断标准 |
|------|------|----------|
| 资产负债率 | 负债水平 | <60% 较安全 |
| 流动比率 | 短期偿债能力 | >1.5 较安全 |

#### 运营能力

| 指标 | 说明 | 判断标准 |
|------|------|----------|
| 存货周转率 | 存货管理效率 | 越高越好 |
| 应收账款周转率 | 收款效率 | 越高越好 |

### 2. 行业地位

- 行业市占率
- 行业排名
- 竞争优势（品牌、技术、渠道等）

### 3. 成长性

- 近 3 年营收复合增长率
- 未来 3 年营收增长预期
- 新业务/新产品进展

### 4. 估值水平

| 指标 | 说明 | 判断标准 |
|------|------|----------|
| PE (市盈率) | 股价/每股收益 | 与行业平均对比 |
| PB (市净率) | 股价/每股净资产 | <3 较安全 |
| PEG | PE/增长率 | <1 低估，>1 高估 |

---

## 分析方法

### 杜邦分析法

```
ROE = 净利率 × 资产周转率 × 权益乘数

通过分解 ROE，分析盈利来源：
- 净利率高 → 产品溢价能力强
- 资产周转率高 → 运营效率高
- 权益乘数高 → 财务杠杆高
```

### 估值对比

```
1. 与自身历史估值对比
2. 与同行业公司对比
3. 与市场平均水平对比
```
