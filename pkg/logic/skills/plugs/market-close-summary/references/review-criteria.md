# 闭市复盘标准

## 复盘评估框架

---

## 1. 预测准确率评估

### 早盘预测评估
| 评估项 | 标准 |
|--------|------|
| 方向判断 | 预测涨跌方向是否正确 |
| 关键价位 | 支撑阻力位判断是否准确 |
| 操作建议 | 建议的操作是否合理 |

### 午盘预测评估
| 评估项 | 标准 |
|--------|------|
| 方向判断 | 预测涨跌方向是否正确 |
| 买入时机 | 建议的买入时机是否合适 |
| 风险提示 | 风险提示是否到位 |

---

## 2. 操作质量评估

### 正确操作识别标准
| 标准 | 说明 |
|------|------|
| 符合策略 | 操作符合用户投资策略 |
| 执行到位 | 决策与执行一致 |
| 收益贡献 | 操作对收益有正向贡献 |
| 风险控制 | 有效控制了风险 |

### 待改进操作识别标准
| 标准 | 说明 |
|------|------|
| 时机偏差 | 买卖时机不够理想 |
| 执行偏差 | 决策与执行有偏差 |
| 信息不足 | 决策依据不够充分 |

### 错误操作识别标准
| 标准 | 说明 |
|------|------|
| 违背策略 | 操作违背用户策略 |
| 程序错误 | 执行顺序或金额错误 |
| 遗漏执行 | 已决策但未执行 |

---

## 3. 收益归因分析

### 收益来源
| 来源 | 计算方式 |
|------|----------|
| 持仓收益 | (收盘价 - 成本价) × 持仓数量 |
| 交易收益 | 卖出金额 - 买入成本 |
| 手续费支出 | 交易次数 × 5 元 |

### 收益率对比
| 对比项 | 说明 |
|--------|------|
| 对比大盘 | 沪深300 涨跌幅 |
| 对比行业 | 所属行业板块涨跌幅 |
| 对比目标 | 用户预期收益目标 |

---

## 4. 风险暴露评估

### 当前风险点
| 风险类型 | 评估内容 |
|----------|----------|
| 仓位风险 | 仓位是否过重 |
| 集中度风险 | 持仓是否过于集中 |
| 行业风险 | 行业暴露是否过大 |
| 流动性风险 | 换手率是否过低 |

### 止损检查
| 检查项 | 标准 |
|--------|------|
| 盈利持仓 | 是否需要移动止盈 |
| 亏损持仓 | 是否触及止损位 |
| 高风险持仓 | 是否需要减仓 |

---

## 5. 经验沉淀标准

### 值得记录的经验
| 类型 | 标准 |
|------|------|
| 新发现规律 | 市场规律、板块联动等 |
| 策略优化 | 策略参数调整建议 |
| 教训总结 | 避免再次犯错的方法 |

### 不需要记录的内容
- 常规操作
- 已有策略覆盖的内容
- 偶然性事件
