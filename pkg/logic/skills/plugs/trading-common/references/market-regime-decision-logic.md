# 市场状态差异化决策逻辑

## 概述

根据 market-regime-classifier 输出的市场状态，调整买卖决策的松紧程度。牛市中放宽买入条件、收紧卖出条件；熊市中收紧买入条件、放宽卖出条件。

---

## 牛市决策逻辑 (强牛/牛/弱牛)

### 买入条件放宽

| 调整项 | 默认规则 | 牛市调整 |
|--------|----------|----------|
| 技术信号要求 | 至少 2 个信号共振 | 至少 1 个强信号即可 |
| 涨幅容忍度 | < 3% | 强牛 5%, 牛 5%, 弱牛 3% |
| PE 估值容忍 | 不高于行业均值 20% | 不高于行业均值 30% |
| 回调买入 | 回调至支撑位 | 回调至 5 日均线即可 |
| 仓位使用率 | 首次 50% | 首次 60% |

### 卖出条件收紧

| 调整项 | 默认规则 | 牛市调整 |
|--------|----------|----------|
| 止盈目标 | +8% 卖 50% | +12% 卖 50% |
| 技术走弱容忍 | 跌破支撑即卖 | 跌破支撑后再观察 1 日 |
| 移动止盈 | 回撤 5% 止盈 | 回撤 8% 止盈 |
| 分批卖出比例 | 首次 50% | 首次 30% |

### 牛市核心原则
```
1. 让利润奔跑 (Let profits run)
2. 不轻易止盈
3. 回调是加仓机会
4. 关注板块轮动，不追已大涨的板块
```

## 熊市决策逻辑 (弱熊/熊/强熊)

### 买入条件收紧

| 调整项 | 默认规则 | 熊市调整 |
|--------|----------|----------|
| 技术信号要求 | 至少 2 个信号共振 | 至少 3 个信号共振 + 基本面支撑 |
| 涨幅容忍度 | < 3% | 弱熊 < 2%, 熊/强熊 < 1% |
| PE 估值容忍 | 不高于行业均值 20% | 必须低于行业均值 |
| 买入时机 | 出现信号即买 | 仅在尾盘确认后买入 |
| 仓位使用率 | 首次 50% | 弱熊 25%, 熊 20%, 强熊 不新增 |

### 卖出条件放宽

| 调整项 | 默认规则 | 熊市调整 |
|--------|----------|----------|
| 止盈目标 | +8% 卖 50% | +5% 卖 50% |
| 技术走弱 | 跌破支撑卖 | 任何技术信号走弱即卖 30% |
| 止损线 | -20% | -15% (弱熊), -10% (熊/强熊) |
| 持股时间 | 无限制 | 持仓不超过 5 个交易日 |
| 隔夜仓位 | 无限制 | 收盘前减仓至 50% 以下 |

### 熊市核心原则
```
1. 现金为王 (Cash is king)
2. 快进快出，不恋战
3. 止损比止盈更重要
4. 午后 (14:00 后) 不新开仓
```

## 震荡市决策逻辑 (震荡偏牛/震荡偏熊)

### 买入条件

| 调整项 | 默认规则 | 震荡市调整 |
|--------|----------|-----------|
| 技术信号要求 | 至少 2 个信号共振 | 至少 2 个信号共振 (不变) |
| 涨幅容忍度 | < 3% | < 3% (不变) |
| 买入时机 | 出现信号即买 | 回调至支撑位才买 |
| 仓位使用率 | 首次 50% | 偏牛 40%, 偏熊 30% |

### 卖出条件

| 调整项 | 默认规则 | 震荡市调整 |
|--------|----------|-----------|
| 止盈目标 | +8% 卖 50% | +5% 卖 50% (快进快出) |
| 技术走弱 | 跌破支撑卖 | 跌破支撑卖 (严格执行) |
| 持仓周期 | 无限制 | 3-5 个交易日 |

### 震荡市核心原则
```
1. 高抛低吸，不追涨杀跌
2. 降低单笔盈利预期
3. 控制仓位等待方向选择
4. 关注突破/跌破信号，准备切换到牛/熊模式
```

## 使用方式

在每次决策前加载本文件，结合 market-regime-classifier 的输出：

```
1. 加载 market-regime-classifier.md → 输出市场状态
2. 加载本文件 → 获取该状态下的差异化参数
3. 使用调整后的参数进行买卖条件判断
4. 在推理链第5部分记录市场状态及其对决策的影响
```
