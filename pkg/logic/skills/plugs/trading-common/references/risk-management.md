# 风险管理规则 (v2.3.0)

> Phase 2 策略体系升级 — ATR 动态止损、回撤熔断、凯利公式

## ATR 动态止损

### 计算方法
1. 使用 `get_stock_history_k(stock_code, period="day", count=14, adjust="qfq")` 获取14天日K线
2. 计算 True Range (TR):
   - TR = max(当日最高-当日最低, |当日最高-前日收盘|, |当日最低-前日收盘|)
3. 计算 ATR(14) = TR 的14日指数移动平均 (EMA)
   - 初始 ATR = 14日 TR 的简单平均
   - EMA_TR = (TR × 2/(14+1)) + (前日ATR × 13/15)

### 初始止损
- 初始止损 = 入场价 - 2 × ATR(14)
- 止损位取整到合理价位（保留2位小数）

### 移动止盈 (Trailing Stop)
- 移动止损位 = 持仓期间最高价 - 1 × ATR(14)
- **止损位只能上移，不能下移** (ratchet 机制)
- 每天重新计算 ATR 和止损位
- 当日最低价触及止损位 → 立即执行卖出

### 降级处理
- 如历史数据不足14天: 使用可用天数计算简单平均 TR 作为近似 ATR
- 在推理链中标注 "ATR 数据不足，使用近似值（N天）"

## 最大回撤熔断

| 级别 | 触发条件 | 措施 |
|------|----------|------|
| 日内 | 当日峰值回撤 > 5% | 暂停所有开仓 |
| 周内 | 周内峰值回撤 > 10% | 减仓至 30% 以下 |
| 月内 | 月内峰值回撤 > 20% | 暂停交易，全面复盘 |

回撤计算: (峰值净值 - 当前净值) / 峰值净值 × 100%

## 时间止损

- 持仓 N 日未达预期收益: 减半仓 (N = 基于交易计划的天数，短线默认 5 天)
- 持仓 2N 日仍亏损: 清仓
- 在推理链中标注持仓天数

## 凯利公式参考仓位

```
凯利仓位 = (胜率 × 赔率 - 失败率) / 赔率
```
- 胜率 = 基于历史同类信号的成功率（默认 0.55）
- 赔率 = 预期收益 / 预期亏损（基于 ATR 止损位计算）
- 失败率 = 1 - 胜率
- **凯利仓位仅作为参考上限**，不超过市场状态限制和固定规则限制

## 使用说明

1. 买入决策后，计算 ATR 初始止损位
2. 每日 review 时更新移动止损位
3. 开仓前检查回撤熔断状态
4. 凯利仓位与固定规则仓位取较小值
