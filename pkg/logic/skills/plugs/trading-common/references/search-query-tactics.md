# 搜索查询策略 - 多角度并行搜索

## 概述

用多组并行查询替代单一关键词搜索，避免算法推荐导致的信息茧房和追热点陷阱。

---

## 核心原则

1. **并行搜索**: 至少同时执行 3 组查询，覆盖不同角度
2. **时间锚定**: 每组搜索必须包含时间限定词
3. **噪音排除**: 主动过滤低质量来源

---

## 3 组并行查询模板

### 查询组 1: 基本面方向
聚焦财务数据、估值、业绩：

```
"[股票名称/代码] 最新财报 业绩 2026"
"[股票名称/代码] PE PB 估值 行业对比"
"[行业名称] 行业分析 政策 2026年"
```

### 查询组 2: 技术面方向
聚焦价格走势、技术指标：

```
"[股票名称/代码] 走势分析 支撑 阻力"
"[股票名称/代码] 资金流向 主力 北向"
"[行业名称] 板块轮动 资金动向"
```

### 查询组 3: 反向/风险方向
主动搜索负面信息和要求：

```
"[股票名称/代码] 风险 利空 负面"
"[股票名称/代码] 减持 监管 问询"
"[行业名称] 风险提示 下行压力"
```

---

## 时间锚定规则

所有搜索查询必须包含时间限定词：

| 场景 | 时间限定词 |
|------|-----------|
| 早盘 (9:30-11:30) | "今日" "今早" "盘前" |
| 午盘 (13:00-14:30) | "午间" "午后" "今日" |
| 收盘 (16:00+) | "今日" "收盘" "盘后" |
| 通用分析 | "近3日" "本周" |

### 示例
```
错误: "贵州茅台 新闻"
正确: "贵州茅台 今日新闻 2026年5月"
正确: "贵州茅台 早盘消息 盘前公告"
```

---

## 噪音排除规则

### 自动过滤以下内容
- 发布时间超过 7 天的旧闻
- 明确标注"广告""推广"的内容
- 来源等级为 X 的内容（见 news-quality-gate.md）
- 纯情绪化标题党（标题含"震惊""引爆""炸裂"且无实质内容）

### 搜索去噪技巧
```
在搜索词中加入排除项:
  -"广告" -"推广" -"荐股" -"喊单"
```

---

## 多角度搜索输出

完成后输出搜索摘要：

```markdown
## 搜索摘要

### 查询组 1: 基本面
- 搜索词: [query]
- 有效结果: [n] 条
- 关键发现: [要点]

### 查询组 2: 技术面
- 搜索词: [query]
- 有效结果: [n] 条
- 关键发现: [要点]

### 查询组 3: 反向/风险
- 搜索词: [query]
- 有效结果: [n] 条
- 关键发现: [要点]

综合信息量: [充分/一般/不足]
```
