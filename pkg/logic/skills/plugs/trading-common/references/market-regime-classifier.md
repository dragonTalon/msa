# 市场状态分类器

## 概述

通过 3 个维度综合判断当前市场状态，动态调整仓位参数。解决牛熊震荡用同一套规则的问题。

---

## 维度 1: 趋势 (Trend)

### 判断指标

| 指标 | 数据来源 | 权重 |
|------|----------|------|
| 大盘指数相对 MA20 位置 | web_search | 30% |
| MA5 与 MA20 交叉方向 | web_search | 25% |
| 近 5 日涨跌家数比 | web_search | 20% |
| 创新高/新低股票比例 | web_search | 15% |
| 连续涨/跌天数 | web_search | 10% |

### 评分规则
```
趋势得分 = Σ(指标值 × 权重)，范围 -100 到 +100

大盘 vs MA20:
  高于 MA20 > 3%:  +30
  高于 MA20 0-3%:  +15
  等于 MA20:        0
  低于 MA20 0-3%:  -15
  低于 MA20 > 3%:  -30

MA5 vs MA20:
  金叉形成中:  +25
  多头排列:    +15
  黏合:         0
  死叉形成中:  -25
  空头排列:    -15

近5日涨跌比:
  上涨 > 60%:  +20
  上涨 50-60%: +10
  上涨 40-50%: -10
  上涨 < 40%:  -20

新高/新低比:
  新高 > 新低:  +15
  持平:           0
  新低 > 新高:  -15

连续走势:
  连涨 ≥ 3天:  +10
  连跌 ≥ 3天:  -10
  其他:           0
```

## 维度 2: 波动率 (Volatility)

### 判断指标

| 指标 | 数据来源 | 权重 |
|------|----------|------|
| VIX/波动率指数 | web_search | 40% |
| 近期日内振幅 | web_search | 30% |
| 涨跌停家数异常 | web_search | 30% |

### 评分规则
```
波动率得分 = Σ(指标值 × 权重)，范围 -100 到 +100
注意: 高波动率 = 高风险 → 负分，推动综合得分向 BEAR 方向

VIX/波动率:
  极低 (低位 20%):  +20  (市场稳定，利好)
  正常 (中位):        0
  偏高 (高位 20%):  -20  (波动大，注意风险)
  极高 (突破上轨):  -40  (恐慌信号)

近5日平均振幅:
  < 1%:     +15  (低振幅，稳定)
  1-2%:       0
  2-3%:     -15  (振幅扩大，风险上升)
  > 3%:     -30  (剧烈波动，高风险)

涨跌停异常:
  无异常:      0
  跌停增多:  -30  (风险信号，推动向 BEAR)
  涨停增多:  +15  (积极信号，但需警惕过热)
```

## 维度 3: 情绪 (Sentiment)

### 判断指标

| 指标 | 数据来源 | 权重 |
|------|----------|------|
| 成交量相对 20 日均量 | web_search | 35% |
| 北向资金流向 | web_search | 35% |
| 融资余额变化 | web_search | 30% |

### 评分规则
```
情绪得分 = Σ(指标值 × 权重)，范围 -100 到 +100

成交量 vs 20日均量:
  先判断位置 (价格 vs MA60):
    低位 (价格 < MA60 × 0.9):  放量 > 1.5倍 = +25 (底部放量，积极)
    中位 (价格在 MA60 ± 10%):  放量 > 1.5倍 = +10 (中性)
    高位 (价格 > MA60 × 1.1):  放量 > 1.5倍 = -25 (高位放量见顶，警惕)
  非放量情况:
    > 1.0-1.5倍:   +10
    0.7-1.0倍:      0
    < 0.7倍 (缩量): -15

北向资金:
  连续3日净流入 > 50亿:  +35
  单日净流入:            +20
  小幅波动:               0
  单日净流出:            -20
  连续3日净流出 > 50亿:  -35

融资余额:
  连续增加:  +15
  持平:        0
  连续减少:  -15
```
> 注: 成交量放量需结合位置判断（价格 vs MA60），详见上方评分规则。

## 数据缺失回退

当 web_search 无法获取某些指标数据时：

### 回退优先级
```
1. 优先使用最近一个交易日的已知数据
2. 缺失维度使用默认值 (0分)，标注"数据不足"
3. 仅 1 个维度可用时 → 输出综合得分 + 标注"低置信度"
4. 0 个维度可用时 → 默认 RANGING_BEARISH (保守)，标注"无法判断"
```

### 最少可用指标
```
每个维度至少需要 2/5 (趋势), 1/3 (波动率), 1/3 (情绪) 个指标有数据
否则该维度标记为"数据不足"，得分为 0
```

---

## 综合状态分类

### 计算综合得分
```
综合得分 = 趋势得分 × 0.5 + 波动率得分 × 0.2 + 情绪得分 × 0.3
范围: -100 到 +100
```
> 置信度标注: 3个维度数据齐全=高置信度, 1-2个维度有数据=中置信度, 仅默认值=低置信度

### 8 种市场状态

| 状态 | 综合得分 | 标签 |
|------|----------|------|
| **强牛** | +60 到 +100 | STRONG_BULL |
| **牛市** | +30 到 +60 | BULL |
| **弱牛** | +10 到 +30 | WEAK_BULL |
| **震荡偏牛** | 0 到 +10 | RANGING_BULLISH |
| **震荡偏熊** | -10 到 0 | RANGING_BEARISH |
| **弱熊** | -30 到 -10 | WEAK_BEAR |
| **熊市** | -60 到 -30 | BEAR |
| **强熊** | -100 到 -60 | STRONG_BEAR |

## 各状态下的参数调整

### 仓位上限

| 状态 | 单只股票仓位上限 | 总仓位上限 | 单次买入上限 |
|------|-----------------|-----------|-------------|
| 强牛 | 总资产 40% | 总资产 90% | 计划仓位 70% |
| 牛市 | 总资产 35% | 总资产 85% | 计划仓位 60% |
| 弱牛 | 总资产 30% | 总资产 80% | 计划仓位 50% |
| 震荡偏牛 | 总资产 25% | 总资产 70% | 计划仓位 40% |
| 震荡偏熊 | 总资产 20% | 总资产 60% | 计划仓位 30% |
| 弱熊 | 总资产 15% | 总资产 50% | 计划仓位 25% |
| 熊市 | 总资产 10% | 总资产 30% | 计划仓位 20% |
| 强熊 | 总资产 5% | 总资产 15% | 不新增买入 |

### 追高限制

| 状态 | 追高阈值 | 说明 |
|------|----------|------|
| 强牛/牛 | 涨幅 > 5% 不买入 | 牛市中允许适度追高 |
| 弱牛/震荡偏牛 | 涨幅 > 3% 不买入 | 默认阈值 |
| 震荡偏熊/弱熊 | 涨幅 > 2% 不买入 | 收紧追高限制 |
| 熊/强熊 | 涨幅 > 1% 不买入 | 熊市严格禁止追高 |

### 分段策略选择

| 状态 | 推荐策略 | 说明 |
|------|----------|------|
| 强牛/牛 | 方式 B (价格信号) | 趋势明确，跟进加仓 |
| 弱牛/震荡偏牛 | 方式 A (时间分段) | 逐步确认 |
| 震荡偏熊 | 方式 C (固定比例) | 保守操作 |
| 弱熊/熊/强熊 | 方式 C + 减仓优先 | 防守为主 |

## 状态转换信号

### 升级信号 (偏熊 → 偏牛)
- 连续 3 日放量上涨
- 北向资金连续 3 日净流入 > 50亿
- 突破关键阻力位
- 融资余额止跌回升

### 降级信号 (偏牛 → 偏熊)
- 连续 3 日缩量下跌
- 北向资金连续 3 日净流出 > 50亿
- 跌破关键支撑位
- 融资余额见顶回落

### 状态确认
```
状态变化需要至少 2 个同向信号确认
单一信号不改变状态，仅标记"观察"
```
