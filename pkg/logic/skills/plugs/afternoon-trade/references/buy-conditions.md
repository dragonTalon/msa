# 午盘买入条件判断规则

## 午盘买入特点

午盘买入更注重：
- 早盘信息的验证
- 午间新闻的解读
- 回调机会的把握

---

## 1. 资金条件

### 基础检查
| 条件 | 要求 |
|------|------|
| 可用余额 | > 0 |
| 已用仓位 | 早盘操作后的最新状态 |
| 剩余可仓位 | 不超过总资产 80% |

### 买入金额计算
```
max_qty = floor(可用余额 / (价格 × 100)) × 100
实际买入 = min(max_qty, 计划仓位 × 50%)
```

---

## 2. 早盘关联分析

### 早盘关注标的
检查早盘 Session 中提到的关注股票：
- 是否出现预期中的回调？
- 是否出现预期中的突破？

### 早盘预测验证
```
早盘预测：[xxx]
上午走势：[xxx]
结论：符合预期 / 不符合预期
```

---

## 3. 午间新闻影响

### 利好消息类型
| 消息类型 | 买入判断 |
|----------|----------|
| 行业利好 | 关注相关板块龙头 |
| 公司利好 | 关注具体公司 |
| 政策利好 | 关注受益板块 |

### 新闻处理流程
```
1. 搜索午间财经新闻
2. 识别利好/利空消息
3. 筛选相关股票
4. 结合技术面判断
```

---

## 4. 技术信号

### 午盘特殊信号
| 信号 | 说明 |
|------|------|
| 早盘回调至支撑位 | 买入机会 |
| 午后放量启动 | 跟进信号 |
| 突破早盘高点 | 趋势确认 |

### 买入时机
- 早盘关注的股票回调至预期价位
- 午间有利好且技术面配合
- 量价配合良好

---

## 5. 时间约束

| 时间段 | 操作建议 |
|--------|----------|
| 13:00-14:00 | 正常买入窗口 |
| 14:00-14:30 | 谨慎买入，小仓位为主 |
| 14:30 后 | 不建议新开仓 |

---

## 午盘买入限制

| 限制 | 说明 |
|------|------|
| 时间限制 | 14:30 后不建议新开仓 |
| 仓位限制 | 首次买入不超过计划仓位 50% |
| 追高限制 | 涨幅 > 3% 不买入 |
| 参考早盘 | 优先关注早盘提及的标的 |
