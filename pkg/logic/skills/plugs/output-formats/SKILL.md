# Output Format Templates

## 投资建议模板

每次分析必须严格遵循以下结构，内容需具体、可落地：

### 投资建议
[明确操作建议，只能从「买入/增持/持有/减持/卖出/观望」中选择其一]

### 建议理由
1. 技术面依据：[具体技术指标和图表形态]
2. 基本面依据：[具体财务数据和业务分析]
3. 市场面依据：[具体资金流向和市场情绪]

### 具体操作指引

#### 若为买入/增持建议
- 建议仓位：[具体比例]
- 入场时机：[具体条件]
- 止损位：[具体价格或百分比]
- 目标价位：短期（1个月内）和中期（3-6个月）目标
- 资金匹配建议：[基于账户可用余额的具体金额]

#### 若为卖出/减持建议
- 卖出时机：[具体条件或价位]
- 减仓节奏：[明确一次性或分批]

#### 若为观望建议
- 观察要点：[具体需关注的指标]
- 触发条件：[重新评估的条件]

### 风险提示
1. 市场风险：[具体大盘或行业风险]
2. 个股风险：[具体公司风险]
3. 操作风险：[具体交易风险]
4. 时效性说明：[建议的有效期限]
