# 早盘新闻谨慎规则

## 概述

隔夜新闻经过一晚发酵，开盘时往往已被定价。早盘使用新闻必须经过额外衰减检查。

---

## 规则 1: 隔夜新闻衰减

### 衰减等级

| 发布时间 | 衰减程度 | 可用性 |
|----------|----------|--------|
| 今日 9:00 后 | 无衰减 | 正常使用 |
| 今日 6:00-9:00 | 轻度衰减 | 可用，降低权重 |
| 昨日 15:00-今日 6:00 | 中度衰减 | 仅作参考，不作为买入主因 |
| 昨日 15:00 前 | 重度衰减 | 大概率已定价，仅作背景 |

### 衰减计算公式
```
新闻权重 = 原始权重 × 衰减系数

衰减系数:
  发布时间 < 2小时:  1.0
  发布时间 2-6小时:  0.7
  发布时间 6-12小时: 0.4
  发布时间 > 12小时: 0.2
```

---

## 规则 2: 开盘跳空检查

### 跳空幅度与已定价判断

```
开盘价 vs 昨收价:
  跳空 < 1%:   正常开盘，新闻可能未被充分定价
  跳空 1%-2%:  部分定价，谨慎使用新闻作为买入依据
  跳空 > 2%:   大概率已充分定价，新闻不作为买入理由
  跳空 > 3%:   过度反应，反而考虑反向操作机会
```

### 跳空 + 新闻关联检查
```
如果 跳空 > 2% 且 跳空方向与新闻方向一致:
  → 标记"新闻已被定价"
  → 禁止基于此新闻同向操作
  → 评估是否有反向交易机会
```

---

## 规则 3: 早盘情绪检查

### 情绪指标

| 指标 | 正常 | 偏热 | 过热 |
|------|------|------|------|
| 涨停家数 (9:35) | < 10 | 10-20 | > 20 |
| 跌停家数 (9:35) | < 5 | 5-10 | > 10 |
| 上涨家数占比 | 40-60% | 60-80% | > 80% |
| 下跌家数占比 | 40-60% | 60-80% | > 80% |

### 情绪极端时的处理
```
如果 上涨家数 > 80% 或 涨停家数 > 20:
  → 市场情绪过热
  → 所有买入决策需额外谨慎
  → 利好新闻的买入权重降低 50%

如果 下跌家数 > 80% 或 跌停家数 > 10:
  → 市场恐慌情绪
  → 评估是否为错杀机会
  → 利空新闻的卖出权重降低 30%（可能已过度反应）
```

---

## 规则 4: 早盘新闻使用决策树

```
获取新闻
  │
  ├─ 发布时间 > 12小时? ──是──→ 仅作背景参考
  │
  ├─ 开盘跳空 > 2%? ──是──→ 新闻已定价，不作为操作主因
  │
  ├─ 市场情绪过热/恐慌? ──是──→ 降低新闻权重 50%
  │
  ├─ 来源等级 < B? ──是──→ 需交叉验证
  │
  ├─ 拥挤度检查 (引用 news-quality-gate.md Step 3)?
  │     ├─ 板块涨幅 > 3% (牛市 > 5%)? ──是──→ 拥挤，谨慎追入
  │     └─ 正常 ──→ 继续
  │
  └─ 通过以上检查 ──→ 正常使用，权重按衰减系数调整
```
