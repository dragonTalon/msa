# 规格：list-knowledge-files

> **能力**: 列出知识文件
> **目的**: 系统必须提供工具来列出知识库目录下的文件，用于 SKILL 查找可用的知识文件。

## 需求：列出知识库文件

系统必须提供工具来列出知识库目录下的文件，用于 SKILL 查找可用的知识文件。

### 场景：列出根目录下的所有子目录
- **当** 调用工具不带参数
- **那么** 返回 `.msa/knowledge/` 下所有子目录名称

### 场景：列出指定目录下的文件
- **当** 调用工具指定目录 `errors`
- **那么** 返回 `.msa/knowledge/errors/` 下所有文件列表

### 场景：列出不存在的目录
- **当** 指定的目录不存在
- **那么** 返回空列表

---

## 需求：路径安全校验

所有路径必须经过安全校验，确保只能访问 `.msa/knowledge/` 目录下。

### 场景：拒绝路径遍历
- **当** 目录参数包含 `..` 或绝对路径前缀
- **那么** 返回错误 "path traversal not allowed"

---

## 需求：文件信息返回

返回的文件列表应包含基本信息。

### 场景：返回文件元数据
- **当** 列出文件成功
- **那么** 返回每个文件的名称、大小、修改时间

### 场景：按修改时间排序
- **当** 返回文件列表
- **那么** 按修改时间倒序排列（最新的在前）
